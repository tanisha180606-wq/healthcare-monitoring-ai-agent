## How It Works

This project is a frontend-only Agentic Healthcare AI demo built using HTML, CSS and JavaScript.

No backend server is required.

## Features

- Diagnostics AI
- Pharmacology AI
- Radiology AI
- Mental Health AI
- Lab AI
- Surgical AI
- Specialist Doctor Recommendation
- Test / Scan Suggestions
- Risk Level Analysis
- Food Guidance
- Safety Warning

## Note

This system gives educational healthcare guidance only and does not replace a doctor.
