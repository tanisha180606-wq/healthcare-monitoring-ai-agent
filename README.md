# HealthMind AI - Agentic Healthcare Support Platform

## Live Website

Open the public website here:

https://cozy-rugelach-a292d8.netlify.app

## Overview

HealthMind AI is a frontend-only Agentic Healthcare AI demo built using HTML, CSS, and JavaScript.

It helps users get basic healthcare guidance by suggesting:

- Risk level
- Specialist doctor
- Tests / scans
- Food guidance
- Safety warning

## AI Agents

1. Diagnostics AI
2. Pharmacology AI
3. Radiology AI
4. Mental Health AI
5. Lab AI
6. Surgical AI

## Features

- Specialist Doctor Recommendation
- Test / Scan Suggestions
- Risk Level Analysis
- Food Guidance
- Safety Warning
- Agent Switching
- New conversation starts when switching agents
- Public website using Netlify

## Technologies Used

- HTML
- CSS
- JavaScript
- Netlify

## How It Works

This project runs fully in the browser.

No backend server is required.

## Disclaimer

This system gives educational healthcare guidance only and does not replace a doctor.
