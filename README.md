# HealthMind AI - Agentic Healthcare Support Platform

## Overview
HealthMind AI is an Agentic AI healthcare system that helps users by:

- Symptom Analysis
- Specialist Doctor Recommendation
- Diagnostic Test Suggestions
- Risk Assessment
- Food Guidance
- Health Awareness

## AI Agents

1. Diagnostics AI
2. Pharmacology AI
3. Radiology AI
4. Mental Health AI
5. Lab AI
6. Surgical AI

## Technologies Used

- HTML
- CSS
- JavaScript
- Python
- Flask
